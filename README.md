## 🧾 What this app does

spanish-tax-calculators is a set of plain JavaScript tools for common Spanish tax tasks. It helps you work out:

- autónomo IRPF
- Social Security for RETA
- ITP
- colaborador planning

It is made for end users who want quick tax figures without using a spreadsheet or manual formulas.

## 🧮 What you can calculate

### Autónomo IRPF
Use this calculator to estimate personal income tax for self-employed work in Spain. It helps you review income, expenses, and expected tax due.

### RETA Social Security
Use this tool to estimate Social Security payments for autónomos. It helps you plan monthly costs and compare payment levels.

### ITP
Use this calculator to estimate Property Transfer Tax for a purchase. It helps you check the tax before you buy.

### Colaborador optimizer
Use this tool to compare options for collaborator work and see which setup fits best for your case.
